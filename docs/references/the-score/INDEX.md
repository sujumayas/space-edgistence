# The Score — visual reference catalog

Frames from the *Amazing World of Gumball* S2E7 **"The Score"** gallery, 
downloaded full-resolution from the Fandom wiki and sorted by relevance to this project.

- Source: https://theamazingworldofgumball.fandom.com/wiki/The_Score/Gallery
- 110 episode frames (folders 01–05) + 17 production design/spritesheet assets (folder `06-designs-spritesheets/`).
- Filenames in 01–05 keep the wiki's `TheScore_NNN.png` numbering.
- The episode's in-show game is literally titled **SPACE EDGISTENCE** (frame 044) — same as this project.
- **The Gallery's "Designs" section (folder 06) is the highest-value art reference** — official Javelle design sheets and animated spritesheets, not screenshots.

## 06-designs-spritesheets/ — Official production design sheets (17)

From the Gallery's **"Designs"** section — Javelle's design art (© 2023 Hanna-Barbera Studios Europe / Warner Bros. Discovery). The richest reference for building actual sprites, tilesets, and UI. Static sheets converted to PNG; animated sheets kept as looping GIF (frame counts noted).

**Sprite & asset sheets**
- `javelle-spritesheet-1.png` — character + town asset sheet: Ocho on bike (multi-angle), neighbours/rivals, skateboard, cars, houses/street tileset, props (→ Rush Hour, School Run)
- `javelle-spritesheet-4.png` — character sprite + UI element sheet
- `javelle-ship-sprites-5.png` — Star Fox-style ship, enemy pyramids, synthwave cockpit background (→ Bills vs Coins, title sequence)
- `spritesheet-animated.gif` — animated sprite sheet, 27 frames (characters, ghosts, coins, maze/board)

**Screen mockups**
- `javelle-screen-mockups-2.png` — 8-bit game screen mockups (match game + others)
- `javelle-screen-mockups-3.png` — gameplay mockups (shooter / space scenes)
- `phone-app-ui-design.gif` — phone-app "relaxation app" UI, animated 93 frames

**Environments**
- `ochos-world-1-level-tileset.png` — side-scroll pixel level / tileset
- `ochos-world-3-apartment-layout.png` — apartment interior line-art layouts (→ Oil Panic, Bug Swatter, Micro Chores)
- `ochos-world-4-apartment-render.png` — apartment interior render
- `ochos-world-5-layout.png` — layout w/ bus + ghost sprites
- `character-turnarounds.png` — Ocho-family character turnaround lineup
- `interior-design.png` — interior (gym/locker) design

- `animation-still-clip-1.gif` / `animation-still-clip-3.gif` — animation stills, 50 / 40 frames
